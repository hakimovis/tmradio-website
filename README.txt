Чтобы вносить изменения в этот сайт, нужно иметь доступ к репозиторию.  Если
он есть, последовательность действий такая:

1. Клонировать репозиторий (создать локальную копию) командой:
   git clone git@github.com:umonkey/tmradio.git
   После этого появится папке tmradio/ с копией репозитория, код сайта
   находится в папке src/, исходный код страниц — в src/input/.

2. Изменить существующие или добавить новые страницы.  Они хранятся в папке
   src/input/ и имеют расширение md (markdown).  Каждая страница начинается с
   параметров, за которыми следует текстовая часть.  Для ознакомления с
   форматом проще всего посмотреть или скопировать какую-нибудь существующую.

3. Для обновления сайта нужно выполнить команду "make update" находять в папке
   src (обновляются html-файлы в корне сайта).  Для проверки результата можно
   выполнить команду "make serve", которая запускает локальный веб-сервер на
   порту 8080.

4. Для обновления сайта с отправкой изменений наружу нужно выполнить команду
   "make update".  Она регенерирует страницы, фиксирует (коммитит) все
   изменения и отправляет из в GitHub.

Обычно сеанс работы выглядит примерно так:

$ cd ~/src/sites/tmradio.net/src
$ cp input/sosonews-07.md input/sosonews-08.md
$ vim input/sosonews-08.md
$ make update
