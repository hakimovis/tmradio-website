Программа для прослушивания радио

Клиент tmradio позволяет слушать радио, просматривать новости (тэг #tmradio в Твиттере), оценивать играющую музыку и обмениваться сообщениям в чате с другими пользователями (всё через jabber).

Сейчас существует версия только для ОС Linux, которую мы планируем портировать под Windows и Mac OS X, также есть планы написать такое же приложение для iPhone и устройств с ОС Android.  Если умеете программировать на чём-нибудь из этого и можете пожертвовать немного своего времени, было бы здорово; можем в ответ пожертвовать немного денег.

Информация об обновлениях

  • Вышел tmradio-client v0.11 для Linux 01.02.2011 комментировать

Версия для Linux

Программа написана на питоне с использованием GTK+, звук проигрывается через Gstreamer.

Текущая версия: 0.11 (zip, deb).
